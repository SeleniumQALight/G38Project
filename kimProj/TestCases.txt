# 1
Пользователь может авторизоваться
1. Открыть страницу Авторизации
2. Ввести login : Student
3. Ввести Пароль : "909090"
4. Нажать "Вход"
ОР: Пользователь авторизован

# 2
Авторизованный пользователь может редактировать свой профиль
1. Открыть страницу Профиль
2. Поменять значения полей: Адрес почты, Пользователь
3. Нажать кнопку "Сохранить"
ОР: Профиль изменен

# 3
Пользователь может добавить Сотрудника
1. Открыть страницу Словари - Сотрудники
2. Нажать "+"
3. Заполнить все поля
4. Нажать "Создать"
ОР: Пользователь на странице Словари- Сотрудники, новый сотрудник удален

#4
Пользователь может удалить Сотрудника
1. Открыть страницу Словари - Сотрудники
2. Открыть страницу редактировани Сотрудника
3. Нажать кнопку "Удалить"
